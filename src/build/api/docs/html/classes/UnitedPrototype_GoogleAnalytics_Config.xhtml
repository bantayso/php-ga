    Members

    protected

    • $anonymizeIpAddresses — UnitedPrototype\GoogleAnalytics\bool
      Whether to anonymize IP addresses within Google Analytics by stripping the last IP address block, will be mapped to "aip" parameter
    • $endPointHost — string
      Google Analytics tracking request endpoint host. Can be set to null to silently simulate (and log) requests without actually sending them.
    • $endPointPath — string
      Google Analytics tracking request endpoint path
    • $errorSeverity — int
      How strict should errors get handled? After all, we do just do some tracking stuff here, and errors shouldn't break an application's functionality in production.
    • $fireAndForget — UnitedPrototype\GoogleAnalytics\bool
      Whether to make asynchronous requests to GA without waiting for any response (speeds up doing requests).
    • $loggingCallback — \Closure
      Logging callback, registered via setLoggingCallback(). Will be fired whenever a request gets sent out and receives the full HTTP request as the first and the full HTTP response (or null if the "fireAndForget" option or simulation mode are used) as the second argument.
    • $requestTimeout — float
      Seconds (float allowed) to wait until timeout when connecting to the Google analytics endpoint host
    • $sendOnShutdown — UnitedPrototype\GoogleAnalytics\bool
      Whether to just queue all requests on HttpRequest::fire() and actually send them on PHP script shutdown after all other tasks are done.
    • $sitespeedSampleRate — int
      Defines a new sample set size (0-100) for Site Speed data collection.

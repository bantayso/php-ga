    Members

    protected

    • $category — string
      Variations on an item, e.g. "white", "black", "green" etc., will be mapped to "utmiva" parameter
    • $name — string
      Product Name, e.g. "T-Shirt", will be mapped to "utmipn" parameter
    • $orderId — string
      Order ID, e.g. "a2343898", will be mapped to "utmtid" parameter
    • $price — float
      Unit Price. Value is set to numbers only (e.g. 19.95), will be mapped to "utmipr" parameter
    • $quantity — int
      Unit Quantity, e.g. 4, will be mapped to "utmiqt" parameter
    • $sku — string
      Product Code. This is the sku code for a given product, e.g. "989898ajssi", will be mapped to "utmipc" parameter

    • $action — string
      The action for the event (e.g. "Play").
    • $category — string
      The general event category (e.g. "Videos").
    • $label — string
      An optional descriptor for the event (e.g. the video's title).
    • $noninteraction — UnitedPrototype\GoogleAnalytics\bool
      Default value is false. By default, event hits will impact a visitor's bounce rate.
    • $value — int
      An optional value associated with the event. You can see your event values in the Overview, Categories, and Actions reports, where they are listed by event or aggregated across events, depending upon your report view.
